FAQ

Frequently Asked Questions

How long does the VPN configuration last?

All VPN packages are valid for 30 days from the date of activation. After 30 days, you need to renew your package to continue using the service.

Do I need an active SIM data package?

Yes, you must already have a supported SIM data package activated on your device. Our VPN configurations work on top of your existing data package.

What does the Unlimited package include?

The Unlimited package (Rs.500) has no GB limits and no speed limits. You can use as much data as you want at maximum speed throughout the 30-day period.

Why do Mobitel users need a Device ID?

Mobitel 222 Zoom package requires HTTP Injector app, which uses a unique Device ID to generate and apply the VPN configuration. This is only needed for Mobitel users.

How do I make the payment?

You can pay via Sampath Bank transfer or Ezcash. If using Ezcash, please add Rs.20 extra. After payment, upload the slip through the order form and submit.

My SIM package is not listed. Can I still use the VPN?

Custom configurations may still be available for unlisted packages, but speed limitations can occur. Contact us via WhatsApp to check compatibility before ordering.
